# Trabalho_BDD

Calculadora web desenvolvida para a atividade de BDD.

## Como executar

Abra o arquivo `index.html` no navegador.

## Testes BDD

Os cenarios em Gherkin ficam em `features/calculadora.feature`.

Para instalar as dependencias:

```bash
npm install
```

Para executar os testes:

```bash
npm run test:bdd
```

No PowerShell do Windows, se o comando `npm` for bloqueado, use:

```bash
npm.cmd install
npm.cmd run test:bdd
```
